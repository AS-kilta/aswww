The template is rendered inside a View object, so $this refers to a View.

Forms

formStart($action)

Start a form with $this->formStart() so the <form action='...'> is filled automatically. formStart() takes the desired method as parameter. For example, formStart('updatePage') would cause renderUpdatePage to be called, when the form is submitted. If nothing is specified, the renderDefault is called.

Example:

<?= $this->formStart('login') ?>
<?= $this->formEnd() ?>
